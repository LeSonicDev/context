# 词汇表
#### 第一組：高頻代詞與虛詞（決定句子結構）

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**1. 之**|往、去（動詞）|**送孟浩然「之」廣陵**|當後面接地點時，它就是動詞 GO。|
|**2. 之**|的（助詞）|**赤子「之」心**|最常見用法。|
|**3. 之**|他/它（代詞）|**學而時習「之」**|指代前面提到的事物。|
|**4. 其**|他的/那（代詞）|**「其」人其事** / 工欲善其事|等於英文的 his/her/its/that。|
|**5. 汝 / 爾**|你|**「汝」安敢輕吾射**（賣油翁）|「爾」虞我詐（你騙我，我騙你）。|
|**6. 吾 / 余**|我|**「吾」日三省吾身**|還有「予」也是我。|
|**7. 諸**|眾、各（形容詞）|**「諸」位同學**|有時是「之於」的合音（付諸流水）。|
|**8. 俱**|全、都、一起|**萬念「俱」灰** / 與生俱來|所有人一起，全部。|
|**9. 或**|有的人（代詞）|**「或」以為死，或以為亡**|**千萬別解作「或者」！**這是大坑。|
|**10. 蓋**|原來是/大概|**「蓋」一癩蝦蟆也**|發語詞，放在句首解釋原因。|

#### 第二組：常見單字動詞（解題關鍵）

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**11. 走**|跑|**「走」馬看花** / 奔走相告|古文的走是 Run，行才是 Walk。|
|**12. 去**|離開|**大勢已「去」** / 揚長而去|**不是 Go，是 Leave！**這是中一最大陷阱。|
|**13. 亡**|逃跑、丟失|**「亡」羊補牢**|不一定是死，更多時候是逃跑或丟了東西。|
|**14. 詣**|拜見|**「詣」太守**（桃花源記）|下級去見上級。|
|**15. 歸**|返回、歸還|**完璧「歸」趙**|女子出嫁也叫「歸」。|
|**16. 反**|返回（通返）|**流連忘「反」**|通假字，看到「反」先想「返」。|
|**17. 辭**|推辭、告別|**不「辭」而別**|也有「言辭」的意思。|
|**18. 欲**|想要|**隨心所「欲」** / 欲速不達|Want。|
|**19. 謂**|說、對...說|**此之「謂」也**|謂...曰 = 對...說。|
|**20. 具**|準備、全|**「具」備** / 百廢具興|通「俱」，全部；或詳細地（具告之）。|
|**21. 顧**|看、回頭|**「顧」影自憐** / 三顧茅廬|回頭看。|
|**22. 嘗**|曾經|**未「嘗」識書具**|嘗試是試，嘗是曾經。|
|**23. 患**|擔心、憂慮|**「患」得「患」失**|也有禍害的意思（水患）。|
|**24. 師**|學習、軍隊|**「師」夷長技** / 班師回朝|既是老師，也是向...學習，還是軍隊。|
|**25. 謝**|道歉、推辭|**閉門「謝」客** / 負荊請罪|不一定是Thank you，常是「拒絕」或「認錯」。|

#### 第三組：形容詞與副詞（決定語氣）

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**26. 甚**|很、非常|**欺人太「甚」**|程度副詞。|
|**27. 益**|更加|**精「益」求精** / 老當益壯|也有「好處」之意（開卷有益）。|
|**28. 固**|本來、堅持|**「固」執己見** / 根深蒂固|固國不以山溪之險（鞏固）。|
|**29. 素**|向來、平時|**「素」不相識**|平時就不認識。|
|**30. 卒**|終於、結束|**「卒」之** / 生卒年|還有士兵的意思（小卒）。|
|**31. 徐**|慢慢地|**清風「徐」來**|徐徐圖之。|
|**32. 曾**|曾經 / 竟然|**似「曾」相識**|讀 zēng 是曾經；讀 céng 通「層」。|
|**33. 殆**|危險、大概|**知己知彼，百戰不「殆」**|不會危險/失敗。|
|**34. 類**|像|**畫虎不成反「類」犬**|類似。|
|**35. 善**|擅長、好|**「善」解人意** / 奕秋通國之善奕者|不只是善良，更多指「擅長」。|
|**36. 雖**|即使|**「雖」千萬人吾往矣**|即使人很多我也要去。轉折連詞。|
|**37. 遂**|於是、就|**半身不「遂」**（讀音例外）/ 遂意|事情做成了，接著做下一步。|
|**38. 方**|正當、剛|**「方」興未艾** / 血氣方剛|正在那個時候。|
|**39. 乃**|於是、才|**家祭無忘告「乃」翁**（你的）|也有「是」的意思（失敗乃成功之母）。|
|**40. 且**|將要、而且|**年「且」九十**（愚公移山）|將近/將要。|

#### 第四組：古今異義與特殊名詞（陷阱區）

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**41. 妻子**|妻 + 子女|**率「妻子」邑人來此絕境**|現代專指太太，古代是老婆孩子。|
|**42. 丈夫**|男子漢|**「丈夫」亦愛憐其少子乎**|現代指老公，古代指成年男子。|
|**43. 幾**|幾乎 / 多少|**「幾」乎** / 所剩無幾|讀 jī 是幾乎；讀 jǐ 是多少。|
|**44. 絕**|斷絕、極高|**「絕」處逢生** / 拍案叫絕|橫渡（假舟楫者，非能水也，而絕江河）。|
|**45. 鄙**|淺陋、邊遠|**肉食者「鄙」**（曹劌論戰）|目光短淺。|
|**46. 書**|寫 / 信 / 書|**家「書」抵萬金**|常常指「信件」或動詞「寫」。|
|**47. 獄**|案件|**小大之「獄」**|不是監獄，是官司/案件。|
|**48. 兵**|兵器 / 軍隊|**短「兵」相接** / 紙上談兵|最初指兵器，後指軍隊。|
|**49. 左右**|侍從|**傳以示美人及「左右」**|不是方向，是身邊跟著的人。|
|**50. 尋**|不久|**未果，「尋」病終**（桃花源記）|作為時間副詞，指「不久之後」。|

### 第五組：一詞多義與高頻動詞（考卷上的「變色龍」）

這組詞最容易讓學生混淆，必須結合上下文判斷。

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**51. 道**|道路 / 道理 / 說|**說三「道」四** / 任重「道」遠|名詞是路/道理，動詞是說（云）。|
|**52. 食**|吃 / 餵養（飼）|**解衣推「食」** / **「食」馬者**（馬說）|讀 sì 時通「飼」，指餵養。|
|**53. 聞**|聽說 / 氣味 / 名聲|**百「聞」不如一見** / 臭不可「聞」|鼻子聞還是耳朵聽？看搭配。|
|**54. 易**|容易 / 交換 / 改變|**以羊「易」之** / 移風「易」俗|看到「以...易...」就是交換。|
|**55. 當**|應當 / 抵擋 / 判罪|**銳不可「當」** / 一夫「當」關|還有「在...時候」（當是時）。|
|**56. 許**|答應 / 左右（數量）|**以身相「許」** / 潭中魚可百「許」頭|放在數字後表示「大約」。|
|**57. 安**|怎麼 / 哪裡 / 安全|**燕雀「安」知鴻鵠之志** / 隨遇而「安」|疑問副詞（How/Where）最常考。|
|**58. 簡**|選拔 / 簡陋 / 竹簡|**「簡」拔人才** / 居心「簡」樸|動詞是挑選，名詞是書冊。|
|**59. 曉**|天亮 / 明白 / 告知|**家喻戶「曉」** / 春眠不覺「曉」|早上天亮，引申為「明白」。|
|**60. 備**|完備 / 詳細 / 準備|**德才兼「備」** / 有「備」無患|形容詞是全，動詞是準備。|
|**61. 窮**|盡頭 / 困窘 / 追究|**無「窮」無盡** / **「窮」冬烈風**|古代多指「不得志」或「盡頭」，非沒錢（貧）。|
|**62. 達**|顯達 / 通達 / 到達|**「達」官貴人** / 四通八「達」|指事業順利，或路通。|

### 第六組：時間與頻率（敘事邏輯的關鍵）

文言文的時間跳躍很快，掌握這些詞就不會迷失在故事的時間軸裡。

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**63. 旦**|早晨|**通宵「旦」** / 枕戈待旦|太陽從地平線升起。|
|**64. 暮**|傍晚|**朝三「暮」四** / 暮氣沉沉|太陽落入草叢中。|
|**65. 俄**|一會兒 / 突然|**「俄」而雪驟**（詠雪）|表示時間很短。|
|**66. 既**|已經 / ...之後|**「既」生瑜，何生亮**|事情已經發生了。|
|**67. 已**|停止 / 已經|**死而後「已」** / 由來「已」久|最常考「停止」（動詞）。|
|**68. 數**|多次 / 幾個 / 計算|**「數」見不鮮**（shuò） / 不可勝數|讀 shuò 是頻繁；讀 shǔ 是計算。|
|**69. 歲**|年 / 年齡|**「歲」寒三友** / 虛度年「歲」|也有「收成」的意思（歲凶）。|
|**70. 昔**|以前|**今非「昔」比**|講故事開頭常用「昔者...」。|
|**71. 猶**|仍然 / 像 / 還|**記憶「猶」新** / 過「猶」不及|Still / Like。|
|**72. 復**|再 / 又 / 恢復|**死灰「復」燃** / 「復」行數十步|Again。|
|**73. 始**|才 / 開始|**千呼萬喚「始」出來**|副詞「才」（Finally）。|
|**74. 畢**|結束 / 全|**原形「畢」露** / 群賢「畢」至|完成或全部。|

### 第七組：人際互動與稱謂（誰對誰做了什麼）

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**75. 語**|告訴 / 說|**貧者「語」於富者曰** / 不言不「語」|讀 yù，動詞，告訴。|
|**76. 請**|請求 / 請問 / 邀請|**負荊「請」罪** / 戰則「請」從|有時表示「請允許我...」。|
|**77. 謁**|拜見|**「謁」見**|比「詣」更正式的拜見。|
|**78. 延**|邀請 / 延長|**「延」客入座** / 苟「延」殘喘|考試常用「延」= 請（Invite）。|
|**79. 遇**|對待 / 遇到|**冷「遇」** / 知「遇」之恩|怎麼對待某人（Treat）。|
|**80. 遺**|送給 / 遺留 / 丟失|**路不拾「遺」** / **「遺」之千金**（wèi）|讀 wèi 是送給；讀 yí 是遺留。|
|**81. 惡**|討厭 / 壞 / 怎麼|**深「惡」痛絕**（wù） / 窮凶極「惡」|讀 wù 是動詞「討厭」。|
|**82. 憐**|愛惜 / 可憐|**同病相「憐」** / **丈夫亦愛「憐」其少子乎**|古文中多指「愛惜」（Love/Pity）。|
|**83. 客**|門客 / 客人 / 謀士|**反客為主**|古代的「客」常指謀士。|
|**84. 公**|尊稱 / 公正 / 公爵|**「公」亦對此**（指對方）|相當於「先生」或「您」。|

### 第八組：邏輯連接與虛詞（閱讀理解的「關節」）

這部分最難翻譯，但決定了句子的因果轉折。

|**詞彙**|**含義**|**易記上下文 / 成語助記**|**煜澤需注意的點**|
|---|---|---|---|
|**86. 以**|用 / 憑藉 / 因為|**「以」貌取人** / 不「以」物喜|萬能介詞。|
|**87. 而**|並且 / 但是 / 地|**出淤泥「而」不染** / 溫故「而」知新|連詞，表轉折（But）或順接（And）。|
|**88. 因**|於是 / 趁機 / 因為|**「因」勢利導** / **「因」起舞**（沛公）|趁著機會（Take opportunity）。|
|**89. 雖**|即使 / 雖然|**麻雀「雖」小**|區分 Even if 和 Although。|
|**90. 若**|如果 / 像 / 你|**「若」隱「若」現** / 旁「若」無人|若=如（Like/If）。|
|**91. 然**|...的樣子 / 這樣 / 但是|**不以為「然」** / 突「然」|用在詞尾變形容詞（Adverb marker）。|
|**92. 耳**|罷了 / 耳朵|**不過爾「耳」**|句末語氣詞「罷了」（That's all）。|
|**93. 耶**|嗎（疑問）|**是「耶」非「耶」**|句末疑問語氣。|
|**94. 哉**|啊（感嘆/反問）|**何足道「哉」**|表感嘆或反問。|
|**95. 夫**|那 / 發語詞 / 丈夫|**「夫」戰，勇氣也**|讀 fú，放在句首引起議論。|
|**96. 遂**|於是 / 達成|**功成名「遂」**|事情完成了。|
|**97. 及**|等到 / 和|**及時行樂** / **「及」其家穿井**|Wait until / And。|

### 第一篇：求學篇 —— 《煜澤問師》

【針對詞彙】： 代詞（吾/汝/之）、基礎動詞（走/去/師/書/謂）

【趙Sir 導讀】： 這篇模擬了《孟子》或《宋濂·送東陽馬生序》的風格，講述求學的誠意。

> 【原文】
> 
> 昔有少年名煜澤，素（29.向來）有大志。嘗（22.曾經）謂（19.對..說）其（4.他的）友曰：「吾（6.我）欲（18.想要）師（24.學習/拜師）名家，讀萬卷書（46.書/經典）。」友笑之（3.他）。
> 
> 煜澤**遂**（37.於是）**去**（12.離開）鄉，**走**（11.跑）百里，**詣**（14.拜見）名師。師問曰：「**汝**（5.你）來何遲？」對曰：「**患**（23.擔心）學識淺陋，願求**益**（27.更加/進步）。」師奇**之**（3.指這件事），**具**（20.詳細地/準備）以書授之。

**【Alex 的語境記憶點】：**

- **走 vs 去**：看到煜澤「去鄉」（離開家鄉），然後「走百里」（跑了百里路），這兩個動作一串聯，這兩個最易錯的字就記住了。
    
- **師**：這裡是動詞，「師名家」就是「向名家學習」。

### 第二篇：法治篇 —— 《將軍夜獵》

【針對詞彙】： 法律軍事（獄/坐/兵/卒）、動作（反/亡/顧/謝）

【趙Sir 導讀】： 這篇模擬了《史記》的風格，這也是香港中學最常考的「人物傳記」類型，涉及衝突與解決。

> 【原文】
> 
> 一將軍夜獵，誤中左右（49.侍從）。左右傷甚（26.非常），幾（43.幾乎）死。吏執將軍，下獄（47.案件/官司），坐（11.定罪/因為）傷人罪。
> 
> 將軍**亡**（13.逃跑）去，欲**反**（16.返回）營調**兵**（48.軍隊）。客**諫**（表格外：規勸）曰：「**丈夫**（42.男子漢）敢作敢當，豈可**枉**（表格外：扭曲）法？」
> 
> 將軍**顧**（21.回頭看）而嘆曰：「**善**（35.好/對）。」**乃**（39.於是）**歸**（15.返回/自首）**謝**（25.認錯/道歉）罪，**辭**（17.辭去）官。**卒**（30.終於）為民。

**【Alex 的語境記憶點】：**

- **獄/坐**：把將軍抓起來「下獄」（審案），判他「坐」（犯罪），這是香港考卷常見的法律場景。
    
- **亡/反/歸**：先逃跑（亡），想回營（反），最後自首（歸），三個移動動詞的細微差別在故事裡很清楚。
    
- **謝**：將軍最後「謝罪」，絕對不是 Thank you，而是 Apologize。

### 第三篇：哲理篇 —— 《島上漁父》

【針對詞彙】： 虛詞與副詞（蓋/絕/或/類/固/方/且）

【趙Sir 導讀】： 這篇模擬了《桃花源記》或寓言故事，難度最高，考的是那些「沒有實體意義」但決定句意的虛詞。

> 【原文】
> 
> 江中有島，與世隔絕（44.斷絕）。島人類（34.像）古人，不知有漢。或（9.有人）問：「外間樂乎？」
>

#### 「數」的三重境界

在香港中學文言文（特別是中三以上或名校中一拔尖題）中，「數」字有三個層次的考法：

1. **Level 1（基礎 - 動詞）：計算**
    
    - 讀音：shǔ
        
    - 例子：_《與妻書》：「不可**數**計。」_
        
    - 含義：數數。
        
2. **Level 2（進階 - 副詞）：屢次、頻繁**
    
    - 讀音：shuò
        
    - 例子：_《學貴有恆》：「**數**易其師。」_（這是前文表格裡的含義）
        
    - 含義：Often / Frequently。
        
3. **Level 3（高階 - 名詞）：命運、規律** —— **這就是文中「此處補充電義（釋義）」的含義**
    
    - 讀音：shù
        
    - 原文句子：_「得失**數**也。」_
        
    - 語譯：得到還是失去，都是**命運（定數）**啊。
        
    - 含義：Fate / Destiny / Natural Law。
        
    - _名校思辨點：_ 這裡體現了道家「順應自然」的思想，考試時如果把這裡翻譯成「計算」或「頻繁」，整句的哲理就完全錯了。
