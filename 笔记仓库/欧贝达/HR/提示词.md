# 任务
```
为一家中国外贸企业，欧贝达，制定业务员和业务助理（跟单）的职级评定标准。
```

```
业务员的考核包括如下几个方面：
1. 业绩达成
2. 毛利率
3. 客户满意度
4. 新客户开发
5. 客户翻单率
6. 回款率

你分析一下，每个方面都包括哪些关键的行为。比如：为了业绩达成，需要做好哪些关键的事情。
然后，为这些关键的事情设计考核指标。
对于可以定量计算的指标，给出计算公式。

不同的业务员职级的考核指标是一样的，区别是不同职级的考核分数区间不同。

最后生成一个表格：每个指标是一行。每个职级是一列，每个职级的每个指标是打分区间。
```

```
业务助理要考核订单成交过程中从协助业务开发到订单或者项目交付这个过程中的各项能力和指标，涵盖但不限于：
1. 内部流程协调能力
2. 客户满意度。比如：
	1. 客户回复反馈及时性
	2. 客户回复反馈准确率
	3. 回款率
	4. 问题处理周到

你对上述考核方面进行增补和细化。不同的业务助理职级的考核指标是一样的，区别是不同职级的考核分数区间不同。

最后生成一个表格：每个指标是一行。每个职级是一列，每个职级的每个指标是打分区间。
```
